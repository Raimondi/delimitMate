# delimitMate

> Vim plugin for automatic closing of quotes, parenthesis, brackets, etc.

## Installation

- [plug.vim](https://github.com/junegunn/vim-plug)

```vim
Plug 'Raimondi/delimitMate'
```

## Features

### Automatic closing & exiting

| Type        | You get   |
| ----------- | --------- |
| `(`         | `(|)`     |
| `()`        | `()|`     |
| `(<S-Tab>`  | `()|`     |
| `{("<C-G>g` | `{("")}|` |
